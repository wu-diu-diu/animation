请使用 Python + Manim CE 制作一个“天线阻抗失配导致大功率发射机功放烧毁”的工程科普动画。

动画风格要求：

1. 整体风格：
- 工程感
- 科技感
- 深色背景
- 类似 3Blue1Brown + 工业科普风格
- 不要卡通风
- 使用发光效果
- 波形动态流畅

2. 分辨率：
- 1920x1080
- 60fps

3. 动画总时长：
约 60~90 秒。

--------------------------------------------------
【核心目标】
--------------------------------------------------

通过“传输线中的波传播”动态展示：

1. 功放输出射频波
2. 波沿馈线传播
3. 天线阻抗失配
4. 波发生反射
5. 前向波与反射波形成驻波
6. 功放端出现过压
7. MOSFET 超过击穿电压
8. 功放烧毁

重点突出：

- 电磁波反射
- 驻波形成
- 电压峰值升高
- VSWR 增大
- 功放击穿

--------------------------------------------------
【视觉结构】
--------------------------------------------------

屏幕结构：

左侧：
功放模块（PA）

传输线（Transmission Line）

天线（Antenna）

顶部：

背景：
深蓝色科技背景

【Scene 2：正常匹配】
--------------------------------------------------

显示系统：

PA → 50Ω Transmission Line → 50Ω Antenna

顶部显示：

Z0 = 50Ω
ZL = 50Ω
Γ = 0
VSWR = 1.0

创建一条沿传输线向右传播的正弦波。

要求：

1. 波是动态传播的
2. 使用 glow 效果
3. 波颜色：
- 前向波：蓝绿色
4. 波到达天线后逐渐消失
5. 天线出现辐射波纹动画

底部显示：

“所有功率被天线吸收并辐射”

持续约6秒。

--------------------------------------------------
【Scene 3：发生失配】
--------------------------------------------------

天线阻抗突然变化：

ZL = ∞Ω

动画效果：

1. 天线颜色从绿色变红
2. 出现 warning 图标
3. 顶部参数动态变化：

Γ 从 0 增长到 1
VSWR 从 1 增长到 ∞

请动态显示公式：

Γ = (ZL - Z0) / (ZL + Z0)

以及：

VSWR = (1 + |Γ|) / (1 - |Γ|)

公式使用 MathTex。

持续约5秒。

--------------------------------------------------
【Scene 4：反射波形成】
--------------------------------------------------

当波到达天线末端后：

产生反射波。

要求：

1. 创建向左传播的反射波
2. 颜色：
- 反射波：红色
3. 波形与前向波同时存在
4. 明确展示：
- forward wave
- reflected wave

请添加：
箭头方向动画。

顶部新增：

Pf = Forward Power
Pr = Reflected Power

动态显示：

Pr increasing...

持续约8秒。

--------------------------------------------------
【Scene 5：驻波形成】
--------------------------------------------------

核心场景。

要求：

前向波与反射波叠加形成驻波。

请使用：

V(z,t) = Vf cos(ωt-βz) + Vr cos(ωt+βz)

动态生成：

1. 波峰固定
2. 波谷固定
3. 节点与腹点明显

驻波使用：
- 紫色发光

请动态显示：

Vmax
Vmin

以及：

VSWR = Vmax / Vmin

顶部参数持续增加。

持续约10秒。

--------------------------------------------------
【Scene 6：功放端过压】
--------------------------------------------------

镜头缓慢 zoom 到 PA。

要求：

左侧功放附近波峰越来越高。

顶部显示：

持续约8秒。

1. PA 内部出现电弧

4. 输出归零

屏幕出现：

并显示：

P = VI

以及：

P_loss = I²R

最终：

整个系统暗下去。

持续约6秒。
